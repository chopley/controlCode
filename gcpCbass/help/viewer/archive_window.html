The archiving status window

The archiving status window shows the file name of the current archiving file, if any, and the current operating parameters of the archiver. These include the following.

  • Archive directory
    This is the directory in which subsequently opened archive files will be placed.
  • Combine
    The number of snapshots that are being combined per archived record.
  • Max file size
    The number of records to write to the current archive file before closing it and starting a new one.
  • Archive filtering
    When this has the value, on, it means that only marked records are being written to the archive file.
All of these parameters can be changed with the archive control command.

Getting hardcopy of a plot

To transfer the plot to a file that can later be plotted, pull down the File menu of the plot, and select Hardcopy. This will bring up a dialog in which to specify the name of the output file postfixed with a / and the name of the PGPLOT device to use. The most useful possibilities are thus:
  • filename/ps
    Draw a grey-scale postscript plot in landscape mode to a file called filename.
  • filename/vps
    Draw a grey-scale postscript plot in portrait mode to a file called filename.
  • filename/cps
    Draw a color postscript plot in landscape mode to a file called filename.
  • filename/vcps
    Draw a color postscript plot in portrait mode to a file called filename.

Erik Leitch (6-Nov-1999)

Adding a graph to an existing plot

To add a new graph to a plot, either hit the "a" key on the keyboard, while the mouse cursor is within the plot window, or pull down the Configure menu of the plot window, and select Add Graph. In both cases, a graph-configuration dialog will be displayed. When you press the Apply button of this dialog, the graph will be added below the other graphs at the bottom of the plot.
Erik Leitch (4-Nov-1999)

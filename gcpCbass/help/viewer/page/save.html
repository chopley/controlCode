Saving the configuration of a page in a file

To allow a given page to be recreated at a later date, one can save the configuration of the page to a file. There are two ways to do this.
  1. You can save the configuration of the viewer, thus saving the configurations of all its pages and plots in one file. This involves pulling down the File menu of the sptViewer main window and selecting Save configuration file to bring up the "Save configuration" dialog.
  2. Or you can save the configuration of an individual page in a file by pulling down the File menu of the page itself, and selecting Save Page Configuration. This will bring up a dialog in which to specify the name of the output configuration file.
In either case the file that is generated can later be reloaded via the viewer's "load configuration" dialog.

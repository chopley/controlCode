The CBASS viewer program

The cbassViewer program provides array control monitoring and control of the control system. It can also be used to inspect the contents of the archive.

Starting the program

Provided that your path includes the location of the CBASS software, starting the viewer program can be as simple as typing: 
  cbassViewer
Other command-line options are also avaialable for specifying sources of data and configuration files.

The main window

The main window of cbassViewer is split into 4 parts, the main menu bar, the schedule queue window, the archive status window, the log output window, and the command input window. These are documented separately.

Monitoring the state of the CBASS

The antenna control control system samples and archives about 3000 different quantities pertaining to its status. This includes information pertaining to the health of the instrument, environmental information, such as the current weather conditions, and the current time, where the telescope is pointing, and the output data of the correlator. The values of these registers can be monitored both graphically and textually. The former is useful if you need to see how a register is changing as a function of time, or how it changes relative to a another register, whereas the latter is useful for showing the current state of the instrument.

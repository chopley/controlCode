The deicingHeater command (usage)

Context:

The deicingHeater command is intended to be used to control the primary deicing heaters as required. By default the heaters are all set to on when the control system is started.

Examples:

The following example turns off zone 1 heaters 
  deicingHeater 0, off
The following example turns on all heater zones 
  deicingHeater all, on

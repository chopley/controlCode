The encoder_zeros command (usage)

Context:

The encoder_zeros command is intended to be used in the control program startup script $GCP_DIR/control/conf/specific/pointing.init to set the offset which is added to commanded positions just before they are send to the ACU and subtracted from actual positions reported by the ACU, to adjust for the zero point of the encoders.

Examples:

The following example sets the encoder zero points 
  encoderZeros -0.012603, 0, 0

setreg register, value

Change the value of a BICEP register.

Arguments:
Register reg
The name of the register to be changed.
RegValue value
The value to assign to the register. This must be positive.

Example:
The following example changes the temperature of one of receiver one's heaters. Note that array indexes start at 0. 
  setreg rx1.htr_req[0], 1400

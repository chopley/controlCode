The benchOffset command (usage)

Context:

The benchOffset command is intended to be used in observing scripts to set the offset to the default focus position of the optical bench, based on focus observations.

It is added to the reference position, which is set by the benchZeros command.

When this or any other optical bench control command is given, the bench actuators immediately move to the new commanded position, unless disabled via the SCU or emergency stop switches/lockouts.

Examples:

The following example sets the bench offsets 

  benchOffset -8.1,-8.1,-8.1,2.6,2.6,2.2

integrate exponent

Change the hardware integration period to (8.192e-4 x 2^exponent) seconds.

Arguments:
IntTime exponent
The integration period as a power of 2 multiplier of 819.2 us.

Example:
The following example sets the hardware integration time to just under 1.68 seconds. 
  integrate 11

Context:
When the control system is started, the antenna control scanner sets up the system timing generator to integrate every 1.68 seconds. This can be changed using the integrate command. The integration time is specified as a power of two exponent multiplier of the minimum hardware integration time. This results in a physical time of (8.192e-4 x 2^exponent) seconds.

Typical values include:

  • 9 -> 0.42 seconds
  • 10 -> 0.84 seconds
  • 11 -> 1.68 seconds

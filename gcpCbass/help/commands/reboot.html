reboot cpu|bicepAntennaControl|the control program

Restart the control system.

Arguments:
SysType type
The target of the reboot, from:
  • cpu: Reboot to the cpu.
  • bicepAntennaControl: Restart the antenna control control system tasks.
  • the control program: Restart the control program.

Example:
The following example tells the antenna control controller to restart the control system without rebooting the CPU. This is quicker than rebooting, and should result in the same re-initialization. 
  reboot the control program

setAntennaLocation [ant]

Set the location of an antenna relative to the fiducial location set in site.

Mandatory Arguments:
Double north
Double east
Double up
Relative position (in meters) of an antenna. Position is relative to the fiducial location set using site.

Optional Arguments:
Antennas ant
A set of one or more antennas to which the command should be applied.
Default is controlled by the setDefaultAntennas command.

Context:
Absolute antenna locations are specified as a two-step process, by setting an absolute fiducial location using the site command, and a location relative to that fiducial location for each antenna, in a coordinate system whose X-Y axes define the tangent plane at the fiducial location, and whose Z axis points straight up at the fiducial location.

shutdown cpu|bicepAntennaControl|the control program

Halt the control system.

Arguments:
SysType type
The target of the shutdown, from:
  • cpu: Shutdown to the VxWorks boot prompt.
  • bicepAntennaControl: Shutdown the antenna control control system tasks, but leave the VxWorks operating system running.
  • the control program: Shutdown the sun control program.

Example:
The following example tells the antenna control controller to shut down all control system tasks, including itself, but to leave VxWorks running. 
  shutdown the control program

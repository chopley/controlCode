turnPower

Turn power on/off to one or more antennas.

Mandatory Arguments:
SwitchState state
on/off to turn power onn or off.
Optional Arguments:
Integer outlet
The outlet to control (1-4). Setting a value of 0 (the default) sets the state of all the outlets.
Antennas antennas
The set of antennas to which the command should be applied.

Examples: 
# Turn the power strips off for all antennas:

turnPower off

# Turn outlet 1 on for antennas 2 and 3:

turnPower on, outlet=1, ant=ant2+ant3
Context:
Each antenna cabinet contains an ethernet-controllable BayTech power strip to which various peripherals will be attached. This command lets you selectively turn power on or off to outlets on the antenna power strips.

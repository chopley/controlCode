slew_rate [az, el, ant]

Specify a reduced slew rate for one or more telescope axes.

Mandatory arguments:
(None)
Optional Arguments:
SlewRate az
The slew rate of the azimuth drive (10-100 percent)
SlewRate el
The slew rate of the elevation drive (10-100 percent)
Antennas ant
A set of one or more antennas to which the command should be applied.
Default is controlled by the setDefaultAntennas command.

Example:
The following example reduces the slew rate of the deck drive to 50% of its maximum speed. 
  slew_rate dk=50
Context:
While it would be nice always to use the highest slew rate available, it has been found that at low temperatures our deck drive will only run at lower speeds, due to the viscosity of its lubricating oil. This command allows one to reduce the speed to get it working again.

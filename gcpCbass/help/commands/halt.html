halt [ant]

Stop the telescope drives and put on the brakes.

Mandatory arguments:
(None)
Optional Arguments:
Antennas ant
A set of one or more antennas to which the command should be applied.
Default is controlled by the setDefaultAntennas command.

Limitations:
Due to the unbreakable nature of PMAC operations, it isn't possible to interrupt an ongoing slew using this command. A halt issued during a slew operation will be queued until the slew has ended.

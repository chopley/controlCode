The pagerEmailAddress command (usage)

Context:

The pagerEmailAddress command is used to manage the list of email addresses to which pager notifications are sent.

action=add adds email to the list.

action=clear removes all entries from the list.

action=list displays the contents of the list.

Examples:

pagerEmailAddress clear
pagerEmailAddress add, jane.doe@usap.gov
pagerEmailAddress add, john.doe@usap.gov
pagerEmailAddress list

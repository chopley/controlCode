inhibit true|false

Start or stop the data strobe.

Arguments:
Boolean state
This should be true if you want to stop taking data, or false to resume data taking.

Example:
The following example restarts the data strobe, and thus initiates data taking by the antenna control scanner task. 
  inhibit false
Context:
The antenna control scanner takes a snapshot of the BICEP registers whenever an interrupt is received from the system-timing-generator VME board. The inhibit command can be used to pause or resume the delivery of these interrupts. This has the effect of pausing and resuming data taking.

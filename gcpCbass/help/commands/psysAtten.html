psysAtten power, [ant, band]

Set the psys attenuation level

Mandatory Arguments:

Attenuation atten
A valid attenuator setting (0-32 dB, in increments of 0.5 dB).

Optional Arguments:

Antennas ant
A set of antennas for which the output power should be set
Bands band
A set of bands for which the output power should be set

Examples:

To set the attenuation to 10 dB for all antennas, all bands: 
  psysAtten 100
To set the attenuation for ant1, band0, to 10: 
  psysAtten 10, ant=ant1, band=band0
Note:

As of this writing, the XAC code on the modules for bands 12-15 contains a bug which will cause subsequent psys commands to be ignored after issuing this command. To restore the downconverter modules to the power-on state, you will have to reset them using the reset command.

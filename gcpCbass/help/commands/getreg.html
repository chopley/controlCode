RegValue getreg register

Return the value of a BICEP register.

Arguments:
Register reg
The name of the register to be returned.

Example:
The following example returns the value of the first element of the rxsim card's output array. Note that array indexes start at 0. 
  print $getreg(rxsim.output[0])
Context:
This function returns the value of any of the BICEP's registers, as unsigned integers. When the setreg command is used to set the value of a register, the acquired function can be used to check whether the current frame buffer reflects the changed register values before issuing the getreg command.

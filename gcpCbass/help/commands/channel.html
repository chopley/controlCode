channel receivers, bands, on|off

Switch the IF amplifiers of selected correlator channels on or off.

Arguments:
Receivers receivers
The set of receivers to target.
Bands bands
The set of frequency bands to target.
SwitchState state
The desired state of the selected channels: on or off.

Example 1:
The following example turns off the IF amplifiers that connect signals from all receivers except rx1 and rx2 to the correlator of band3. 
 channel all-rx1-rx2, band3, off
Example 2:
The next example turns off all of the IF amplifers that connect receiver 4 to each correlator. 
 channel rx4, all, off

Context:
The BICEP has 13 receivers (rx0..rx12) whose 10GHz bandpasses are split into 10 bands (band0..band9) by a channelizer. The resulting 130 signal paths are refered to as channels. Each of these channels contains an IF amplifier to boost the received signals before feeding them into one correlator module per band. The channelizer provides switches to enable or disable these IF amplifiers. Disabling an IF amplifier effectively disconnects the the input signal from its target correlator input and thus can be used to measure correlator offsets.

Note that all of the IFs are switched on when the system starts.

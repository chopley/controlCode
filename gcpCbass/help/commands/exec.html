The exec command (usage)

Context:

The exec command executes command as a Linux command in a shell. The maximum length of the command is 127 characters.

The output of the command is written to a log file in the directory specified by the scriptDir command.

Commands can be executed on a remote computer using ssh.

Examples:

exec "ssh root@optics /home/zak/sptSecondaryCryostatDaq/main_CalibratorControl.exe -ChopperFreq 19"

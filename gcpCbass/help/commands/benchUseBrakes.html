The benchUseBrakes command (usage)

Context:

The benchUseBrakes command is intended to be used in observing scripts to determine how optical bench position is maintained.

If useBrakes is true, the optical bench actuator brakes are applied and motors turned off when the commanded position is reached. If useBrakes is false, the actuator positions are maintained with a continuous motor servo.

Examples:

The following example enables use of brakes on optical bench actuators 
  benchUseBrakes true

The atmosphere command (usage)

Context:

The atmosphere command sets the current value of temperature in degrees Celcius, relative humidity in percent, and pressure in mmHg. These data are required for the refraction calculations when pointing the telescope.

This command is not normally used by SPT as this information is provided to the control system by the weather station through an RS-232 serial connection to sptcontrol. It may be useful for testing or operation when the weather station is not working.

Examples:

atmosphere -34.0, 674.8, 61

setDefaultAntennas

Specify the default set of antennas.

Mandatory Arguments:
Antennas antennas
A set of one or more antennas to which subsequent commands should be applied.

Examples: 
# Tell only antennas 1 and 2 to track the moon:

setDefaultAntennas ant1+ant2

track moon
Context:
All antenna control commands include an optional parameter specifying which antennas that command should be applied to. When that argument is not specified, the set of antennas defaults to the set defined by this command. On startup of the control system, this set defaults to all antennas.

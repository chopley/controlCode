fg register, value

Write registers on the frame grabber board.

Arguments:
FgReg register
A register of the frame grabber board.
BitMask action
The value to write to the register.

Example 1:
Set the frame grabber to use input look-up table 1: 
  fg incsr1, b0
Example 2:
Invert the frame grabber image: 
  fg incsr2, 0b010000000000
Context:
This function allows the user to control the operation of the frame grabber at a fairly basic level. See FgReg for details.

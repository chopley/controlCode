quad receivers, phase

Configure the quadrature phase shift networks of specified receivers.

Arguments:
Receivers receivers
The set of receivers to target.
QuadPhase phase
The selected quadrature phase (0,90,180 or 270 degrees).

Examples:
The following example adds a 180 degree phase shift to the signals of all receivers except receiver 3 
  quad all-rx3, 180
Context:
The local oscillator that is fed to each receiver goes through a quadrature phase shift network. This can shift the phase of the oscillator by an integer multiple of 90 degrees between 0 and 270 degrees. After mixing with the incoming radio signal this results in an apparent phase shift of the same amount in the output signal.

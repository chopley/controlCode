Hostname hostname()

The name of the computer that is running the control program.

Example:
The following example shows how one could use the hostname constant in the the control program initialization script to issue a warning about the lack of an ifhost clause for the computer that the control program is running on. 
 ifhost bicepnet.ovro.caltech.edu {
   open archive, dir=/scr2/bicepdaq/arc
   open log, dir=/scr2/bicepdaq/log
 } {
   print "I don't know where to put archive and log files on ", $hostname
 }

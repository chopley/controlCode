stow [ant]

Stow the telescope.

Mandatory arguments:
(None)
Optional Arguments:
Antennas ant
A set of one or more antennas to which the command should be applied.
Default is controlled by the setDefaultAntennas command.

Example:
The following example stows antenna 3. 
  stow ant=ant3
Context:
The stow position is currently defined to be the zenith.

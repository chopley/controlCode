optical_camera camera|stepper|focus, count

Control the optical camera and stepper motor.

Arguments:
OpticalCamera camera|stepper|focus
The target of an optical_camera command.
OptCamCount count
An integral number of steps to step the stepper motor, or the words: on or off.

Examples:
The following example steps the motor 200 steps in a counterclockwise direction: 
  optical_camera focus, -200
Context:
This command controls solid-state relays which turn the optical camera and stepper motor on and off via the noise generator digital I/O board, and specifies the number of pulses to be sent to the stepper motor. If the pulse count is positive, the stepper will rotate in a clockwise direction; if negative, the stepper will rotate in a counterclockwise direction.

On power up, both the camera and stepper motor should be turned on. If the noise generator board is unflagged, the camera and stepper motor will be set to the last commanded state.

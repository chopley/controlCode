thermo_readout_interval dt

Set the temperature measurement interval of the thermometry task.

Arguments:
Interval dt
The interval between temperature measurments, or zero to disable the periodic readout facility.

Example 1:
The following example sets the interval between temperature measurements to 1 minute. 
 thermo_readout_interval 1m
Example 2:
The following example stops periodic readouts from being taken, leaving it up to the user to explicitly request individual readouts with thermometer read commands. 
 thermo_readout_interval 0m

Context:
By default, devices attached to a DS2480 line driver are not read out.

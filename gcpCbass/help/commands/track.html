track source, [type, ant]

Start tracking a new source.

Mandatory Arguments:
Source source
A source in the source catalog.
Optional Arguments:
Tracking type
The type of tracking center.
Default is: both
Antennas ant
A set of one or more antennas to which the command should be applied.
Default is controlled by the setDefaultAntennas command.

Examples:
The following example slews the telescope to the current position of 3c273, then procedes to track it. 
  track 3c273
The following example parks the telescope at the zenith. 
  track zenith
Context:
Sources in the source catalog include sidereal J2000 source positions, solar-system objects located via ephemerides, and fixed azimuth, elevation and deck angle pointings. The latter are transparently translated to slew commands.

If while tracking a source, the source falls below the minimum elevation reachable by the BICEP, the control system will continue to track the source in azimuth, but the elevation will be held at its lower limit until the source rises above it.

Antennas can be separately commanded to track different sources. allowing the BICEP to be used as several independent subarrays. If no optional antenna argument is specified, the set of antennas to be commanded defaults to what was set by the last setDefaultAntennas command.

Gross delays are digitally removed in the correlator and fine delays are removed by the lobe rotator, by inserting appropriate phase shifts at the inputs to the correlator for each antenna. Delays are calculated from the observing frequency, and tracking and phase center for each antenna. The type parameter lets you set either the pointing or phase center, or both, for the specified antenna set.

service [ant]

Send the telescope to the service position.

Mandatory arguments:
(None)
Optional Arguments:
Antennas antennas
A set of one or more antennas to which the command should be applied.
Default is controlled by the setDefaultAntennas command.

Example:
The following example sends the antenna 3 to the service position. 
  service ant=ant3
Context:
The service position is currently defined to be AZ = 180, EL = 0

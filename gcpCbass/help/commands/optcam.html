optcam target, action

Control the optical camera, stepper motor, camera control box and frame grabber.

Arguments:
OptCamTarget target
The target of an optcam command.
OptCamCount action
An integral number of steps to step the stepper motor, or a word, specifying the action to apply to the target.

Example 1:
The following example steps the motor 200 steps in a counterclockwise direction: 
  optcam focus, -200
Example 2:
The following example turns the optical camera on: 
  optcam camera, on
Example 3:
The following example sets the shutter speed to 1/100 s: 
  optcam shutter, 100
Example 4:
The following example opens the iris: 
  optcam iris, open
Context:
When the target is camera, stepper, or focus, this command controls solid-state relays which turn the optical camera and stepper motor on and off via the noise generator digital I/O board, and specifies the number of pulses to be sent to the stepper motor. If the pulse count is positive, the stepper will rotate in a clockwise direction; if negative, the stepper will rotate in a counterclockwise direction.

On power up, both the camera and stepper motor should be turned on. If the noise generator board is unflagged, the camera and stepper motor will be set to the last commanded state.

This command can also be used to control various optical camera functions, including the iris, shutter speed, automatic gain control, sensitivity, etc. See OptCamCount for details.

In addition, this command controls the DT1451 frame grabber.

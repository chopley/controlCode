The configureCmdTimeout command (usage)

Context:

The configureCmdTimeout command enables or disables and sets the duration of a timer to abort a read of an interprocess communication channel if the expected information is not received withing the specified time. This prevents the entire control system from hanging if one component crashes or hangs.

If state=on the timeout is enabled. If state=off the timeout is disabled.

Interval is the duration of the timeout in seconds.

Examples:

configureCmdTimeout on, 10  # enable ten second timeout

configureCmdTimeout off     # disable timeout

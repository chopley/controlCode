heater receivers, heaters, volts

Switch off or set the temperatures of one or more receiver heaters.

Arguments:
Receivers ant
The set of receivers whose heaters are to be controlled.
Heaters heaters
The heaters to be controlled. This is a set of one or more of:
  • cryo - The heater in the cryogenic part of the receiver.
  • ambient0 - The first of two ambient temperature heaters.
  • ambient1 - The second of two ambient temperature heaters.
  • all - All of the above heaters.
HeaterVoltage volts
The voltage of the heater between 0 and 10, or the word: off.

Example:
The following example turns off both ambient heaters in receivers 3 and 4. 
 heater rx3+rx4, ambient0+ambient1, off
Example 2:
The following example sets the target sensor voltages of the two ambient temperature controllers to 3.1v. 
 heater rx3, ambient0+ambient1, 3.1

Context:
On power-up the heaters in each receiver are switched off. If they are needed, then they must be turned on and given a target temperature using this command. At the time of writing, none of these heaters are actually used.

grabber combine=number

Configure the frame grabber.

Arguments:
Count combine
The number of snapshots from the frame grabber for the antenna control CPU to average to form the next frame grabber image.

Example 1:
The following example tells the frame grabber to combine 10 successive images before returning the result. 
  grabber combine=10

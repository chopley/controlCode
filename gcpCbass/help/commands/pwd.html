pwd

Display the current working directory.

Arguments:
(none)

Example:
The following example, shows the current working directory before and after using the cd command to change it. 
  pwd
            -> /home/goblin/bicepdaq/
  cd BICEP
  pwd
            -> /home/goblin/bicepdaq/BICEP

Context:
The cd and pwd commands act essentially like the unix shell commands of the same name.

When the user doesn't specify a full pathname to a command like schedule, the full pathname is constructed by appending the user's relative pathname to the current working directory. The cd command allows this working directory to be changed, and the pwd command lets one see what it is currently set to.

The collimate_fixed command (usage)

Context:

The collimate_fixed command is intended to be used in the control program startup script $GCP_DIR/control/conf/specific/pointing.init to set the collimation parameters of a pointing model. It may also be used at other times to experiment with different pointing models.

Examples:

The following example sets the collimation parameters 
collimate_fixed optical, -0.026372, -0.581416, ant=ant0
collimate_fixed radio, 0.1717, -0.2222, ant=ant0

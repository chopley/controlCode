The benchZeros command (usage)

Context:

The benchZeros command is intended to be used in observing scripts to set the default focus position of the optical bench. All actuators default to a position of 25 millimeters, approximately at the middle of their useful range, when the control system antenna control program is restarted. When this or any other optical bench control command is given, the bench actuators immediately move to the new commanded position, unless disabled via the SCU or emergency stop switches/lockouts.

Examples:

The following example sets the bench reference position to its power on value. 

  benchZeros 25.0,25.0,25.0,25.0,25.0,25.0

site longitude, latitude, altitude

Set the location of the array. This will be stored as the fiducial location of each antenna. Note that you must ultimately set the actual location of each antenna relative to this fiducial location by using the setAntennaLocation command if you want tracking calculations to be really correct.

Arguments:
Longitude longitude
The geodetic eastward longitude of the site (-180..180 degrees).
Latitude latitude
The geodetic latitude of the site (-90..90 degrees).
Altitude altitude
The geodetic altitude of the site above sea level (meters).

Examples:
The following example would set the BICEP up for operation at the Owen's Valley Radio Observatory, which is 118 degrees, 16 minutes and 56 seconds west of Grenwich, 37 degrees, 14 minutes and 2 seconds north of the equator, and is at an altitude of 1222 meters above sea level. 
  site -118:16:56, 37:14:02, 1222.0
Context:
Before the control system can track a source, you must call the site command to tell it where the telescope is located.

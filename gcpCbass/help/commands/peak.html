PointingOffset peak(offset)

Return the current peak of a given frame grabber image.

Arguments:
Peak offset
The offset of interest (in degrees).

Example 1:
The following example displays the x-offset of the frame grabber image peak 
  print $peak(x)

Context:
When the frame grabber returns an image, the x and y offsets of the peak pixel are automatically computed. This function returns their value to the scripting interface.

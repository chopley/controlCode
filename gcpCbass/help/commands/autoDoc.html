The autoDoc command (usage)

Context:

This command generates template html help files from the control system source code for all scripting language commands and data types in dir. If dir is not specified, the directory in which the control system was run is used. Two directories, commands and dataTypes, are created within dir.

The following files are created: 

StyleSheet.css
CommandIndex.html
CommandList.html
dataTypes/Index.html

For each command:

  commands/_Synopsis.html
  commands/.html

For each data type:

  dataTypes/_Synopsis.html
  dataTypes/.html
Only the .html and .html files need to be manually edited to include Context and Example information. If these files already exist when the autoDoc command is given, they will not be modified by the command.

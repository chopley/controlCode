coldhead receivers, on|off

Switch the coldheads of selected receivers on or off.

Arguments:
Receivers receivers
The set of receivers to target.
SwitchState state
The desired state of the selected coldheads: on or off.

Example 1:
The following example turns off the coldheads for all receivers except rx1 and rx2. 
 coldhead all-rx1-rx2, off

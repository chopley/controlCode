The benchAcquiredThreshold command (usage)

Context:

The benchAcquiredThreshold command is intended to be used in observing scripts to set the "dead band" distance from the commanded position within which each optical bench actuator is considered to be on target.

If optical bench brakes are enabled, when an actuator gets within threshold millimeters of its commanded position, it is stopped and the brake is applied.

Whether or not brakes are enabled, the expresion $acquired(bench) becomes true only when all actuators are within threshold millimeters of their commanded position.

Examples:

The following example sets the threshold to two hundredths of a millimeter 
  benchAcquiredThreshold 0.02

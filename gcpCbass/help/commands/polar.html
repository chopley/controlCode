polar receivers, left|right, position

Set the encoder position which corresponds to left or right circular polarization for a given receiver.

Arguments:
Receivers ant
The set of receivers whose encoder positions are to be stored.
PolarState state
The polarization state, from
  • left - Left circular polarization.
  • right - Right circular polarization.
PhaseShift position
The target motor encoder position (0..1023). The motor moves a full revolution between 0 and 1024. This corresponds to a phase change of 4.PI.

Example:
The following example sets the encoder position corresponding to left circular polarization for rx9 to 914: 
 polar rx9, left, 914

Context:
In general, each receiver will have a different encoder position which corresponds to left or right circular polarization. By pre-defining these using the polar command, you can subsequently request either state by name when using the pshifter command.

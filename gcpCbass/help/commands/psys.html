psys power, [ant, band]

Set the Psys power level

Mandatory Arguments:

DcPower power
A valid power level

Optional Arguments:

Antennas ant
A set of antennas for which the output power should be set
Bands band
A set of bands for which the output power should be set

Examples:

To set the power for all antennas, all bands, to 100: 
  psys 100
To set the power for ant1, band0, to 100: 
  psys 100, ant=ant1, band=band0
Note:

As of this writing, the XAC code on the modules for bands 12-15 contains a bug which will cause this command to be ignored after a psysAtten command has been issued. To restore the downconverter modules to the power-on state, you will have to reset them using the reset command.

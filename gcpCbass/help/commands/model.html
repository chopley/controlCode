model radio|optical [ant]

Select between the optical and radio pointing models.

Arguments:
Model model
The pointing model to be used (radio or optical).
Antennas ant
A set of one or more antennas to which the command should be applied.
Default is controlled by the setDefaultAntennas command.

Example:
The following example selects the optical pointing model. 
  model optical
Context:
In addition to the BICEP radio receivers, a small optical telescope is mounted on the deck of the BICEP. This is used for measuring the pointing model. Unfortunately it isn't possible to precisely align the optical telescope along the same axis as that of the average radio dish, so it needs a different collimation model. The model command allows one to select between the two pointing models.

Double imstatstat

Return statistics of the latest frame grabber image.

Arguments:
Imstat stat
The statistic of interest.

Example 1:
The following example displays the snr of the frame grabber image 
  print $imstat(snr)

Context:
When the frame grabber returns an image, statistics of the image are automatically computed. This function returns their value to the scripting interface.

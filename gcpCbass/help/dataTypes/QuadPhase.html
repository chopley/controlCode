QuadPhase

One of the 4 quadrature phases 0, 90, 180 or 270.

Input Format:

One of the following numbers.
  • 0 - No phase shift.
  • 90 - 90 degrees phase shift.
  • 180 - 180 degrees phase shift.
  • 270 - 270 degrees phase shift.

Context:
Instantaneous quadrature phase shifts can be inserted by way of a quadrature phase shift network controlled by the quadPhase command.

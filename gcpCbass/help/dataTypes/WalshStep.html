WalshStep

A particular step of a receiver polarization walsh cycle.

Input Format:

An integral value between 0 and 15 (inclusive).

Use with do-loops:
When a WalshStep variable is used as the dependent variable of a do-loop, the associated loop increment is specified as a WalshStep constant.

Examples:
The following example steps receivers rx0 and rx9 through one complete polarization Walsh cycle, integrating for 5 seconds at each step: 
 do WalshStep i=0,15,1 {
   pol_walsh rx0+rx9, half, $i
   until $elapsed() > 5s
 }
Context:
This datatype is used by the pol_walsh command.

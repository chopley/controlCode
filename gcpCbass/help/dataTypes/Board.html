Board

A SPT register-board name.

Input Format:

The name of a board as an unquoted string of alphanumeric characters.

Example:
The following example unflags receiver card 1. 
 unflag rx1
The board names that are currently known include:
  • phase - A phase-shift motor transaction.
  • local - A board of archival configuration registers.
  • tgen - The System Timing Generator board.
  • dio - A currently unused digital I/O board.
  • rx1 - The first receiver card.

Heaters

A set of antenna-based heaters.

Input Format:

A set of one or more of:
  • cryo - A heater in a receiver cryostat.
  • ambient0 - One of two heaters outside the receiver cryostat.
  • ambient1 - The second of two heaters outside the receiver cryostat.
  • all - All of the above heaters.

Examples:
The following example requests that both ambient temperature heaters of receiver 0 be set such that the corresponding thermisters produce an output voltage of 3.03 volts. 
  heater rx0, ambient0+ambient1, 3.03
Context:
Variables of type Heaters are used by the heater() command.

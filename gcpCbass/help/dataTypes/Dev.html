The pathname of a serial device.

Input Format:

An absolute path name. This must name a device to which sptControl has access.

If no device is specified, the device defaults to /dev/cua/a. This argument is relevant only with the pager open command, and is ignored for all other values of the state variable.

Context:
Normally, this will be one of the two available ports on omega0: /dev/cua/a or /dev/cua/b.

Historically, /dev/cua/a has been used as a console to the VME crate, and /dev/cua/b has been used to communicate with the pmac via the pmacterm program.

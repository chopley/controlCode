PowerMeterCommand

The type of command to send to the noise-cal power meter.

Input Format:
  • reset - Reset the power meter.
  • zero - Perform a zero-calibration of the power meter.
  • dbm - Set the power meter to report powers in dBm.
  • read - Take a new power reading without waiting for the next readout interval to expire.

Example:
The following example resets the power meter to its power-on defaults. 
 power_meter reset

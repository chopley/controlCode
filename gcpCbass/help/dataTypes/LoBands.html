LoBands

An LO frequency specifier

Input Format:

One of the following keywords:

    The following are all synonyms for xband:

  • xband
  • loref
  • yig

    The following are all synonyms for kaband:

  • kaband
  • 1cm

    The following are all synonyms for wband:

  • wband
  • 3mm

    This last specifies the highest frequency band:

  • 1mm

Examples:
The following example turns the RF monitor of the YIG oscillator on: 
lo yig, rfmonitor, on
Context:
The lo command requires an LoBands argument.

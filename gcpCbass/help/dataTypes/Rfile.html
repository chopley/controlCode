The Rfile datatype.
Rfile

The pathname of a readable file.

Input Format:

An absolute path name. This must name a file to which the control program has read access.

Examples:
The following example schedules a script to be run. The script resides in a subdirectory of the home directory of a user called foo. 
 schedule ~foo/SPT/schedules/observer.sch

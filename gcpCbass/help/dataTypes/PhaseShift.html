PhaseShift

The target encoder reading of a phase-shift motor.

Input Format:

  • off - Turn the phase shifters off
  • left - The encoder position corresponding to left circular polarization for these receivers (see Context, below)
  • right - The encoder position corresponding to right circular polarization for these receivers (see Context, below)

Use with do-loops:
When a PhaseShift variable is used as the dependent variable of a do-loop, the associated loop increment is specified as an Integer constant.

Context:
Each receiver of the SZA has an associated phase shifter motor. The PhaseShift type is used by the pshifter command to set the target position of a phase-shift motor. The legal range of encoder readings corresponds to a phase shift of approximately 4.PI. When preset encoder positions for left and right circular polarization are defined using the polar command, the words left and right may be used to request these positions.

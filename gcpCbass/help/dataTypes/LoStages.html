LoStages

One or more stages of the local oscillator chain.

Input Format:

A set of one or more of:
  • if - The local-oscillator IF-monitor output.
  • sweep - The sawtooth used to acquire phase-lock.
  • gunn - The gunn-diode.
  • rfmonitor - The RF monitor.
  • all - All of the above stages.

Context:
The lo command requires an LoStage argument.

RxBand

A set of one of more receiver designations.

Input Format:

A set of one or more of:

Equivalent designations for the 30 GHz receiver: 
     rx1cm     
     rx30ghz
Equivalent designations for the 90 GHz receiver: 
     rx3mm   
     rx90ghz
Equivalent designations for the 230 GHz receiver: 
     rx1mm
     rx230ghz
Or the word: 
     spare

Examples: 
# Send the tertiaries to the 30 GHz receiver position:

positionTertiary rx30ghz

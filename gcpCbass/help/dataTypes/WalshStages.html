WalshStages

A specification of walsh-function modulation and/or demodulation stages.

Input Format:

A set of one or more of:
  • mod - The phase-switch modulation function.
  • demod - The phase-switch demodulation function.
  • both - Both the phase-switch modulation and demodulation functions.

Examples:
The first of the following commands sets the phase-switch modulation functions of receivers 3 and 4, to walsh-function 2. The second sets both the modulation, and demodulation functions of all except receivers, 3 and 4, to walsh function 1. 
 walsh mod, rx3+rx4, 2

 walsh both, all-rx3-rx4, 1
Context:
This datatype is only used by the walsh command.

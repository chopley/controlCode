Bands

A set of zero or more SZA frequency bands.

Input Format:

A set of one or more of:

band0 band1 band2 ... band15

Where each member refers to a particular frequency band of the correlator.

Examples:
Enable auto level control for all antennas, bands 4 and 10: 
ifalc on, ant=all, band=band4+band15
Context:
The 8 GHz frequency span of the SZA is separately correlated in 16 adjacent 500 MHz bands. Commands which require a band specification take a Band argument which enables the user to selectively configure one or many bands in one command.

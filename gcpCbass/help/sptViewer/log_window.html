The log window

The log window displays both messages from the control system and some informational and error messages from sptViewer itself. All messages from the control system are prefixed with the current time in UTC. Informational messages are displayed in green, and error messages are displayed in red. Every time that an error message arrives, the viewer emits a beep to warn you to check for error messages in the log window.

Erik Leitch (4-Nov-1999)

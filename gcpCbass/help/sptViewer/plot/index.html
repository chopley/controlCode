Displaying live data graphically

To create a window for displaying live data values graphically, pull down the Configure menu of the main window and select "Add Plot". This will create an empty plot with a menu bar, and display a configuration dialog for setting its initial characteristics.

Plots consist of one or more graphs that share a common X-axis. They are stacked vertically, one above the other. They can either display data within a fixed x-axis range, or be asked to scroll like a chart recorder, to show new data as it arrives. The user can interactively zoom along both the x and y axes, and there are interative options for identifying data points and measuring statistics.

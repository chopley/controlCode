The sptViewer main menu
The menu bar at the top of the main sptViewer window contains 4 menus:
  • The File menu provides all operations that involve files, including loading and saving configuration files, connecting to the control program and reading data from the archive.
  • The Configure menu provides dialogs for manual configuration of the viewer, such as general viewer configuration, adding plots and adding pages.
  • The Utilities menu provides various utilities such as the optical pointing window.
  • The Help menu provides quick access to the help pages that you are currently reading.
Erik Leitch (4-Nov-1999)
